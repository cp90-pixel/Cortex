Cortex Browser

Welcome! Enter a URL or a search term in the navigation bar to start exploring the web. Cortex Browser is a lightweight showcase project built entirely from scratch with Qt and Python.

  • Navigate: Use the toolbar buttons or familiar keyboard shortcuts like Ctrl+L and Enter.
  • Search: Type anything with spaces and Cortex will perform a DuckDuckGo search.
  • Local files: Open HTML files on your computer with the “Open File” action.
  • Reload: Refresh the current page with F5 or the Reload button.
